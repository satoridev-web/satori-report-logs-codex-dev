# SATORI Report Logs — Product Brief (Client-Friendly)

## Summary
SATORI Report Logs provides a monthly service reporting tool for internal teams. It mirrors the structure of Ball Australia’s existing service log format.

## Purpose
To streamline monthly service documentation and generate clean outputs for management review.

## Key Features
- Monthly service log creation
- Structured fields based on Ball template
- Clean exports (HTML, CSV, PDF)
- Printable layouts
- Admin-friendly interface

## Report Sections
- Service Details
- Overview
- Tasks Completed
- Issues Found
- Notes
- Sign-off

## Workflow
1. Select month
2. Enter data in the editor
3. Preview
4. Export or print

## Users
- Admin staff
- Service/maintenance technicians
- Managers reviewing reports

## Roadmap
- Email delivery
- Multi-site support
